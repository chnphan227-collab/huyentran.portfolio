Data Analytics Project

AIRBNB MARKET INSIGHTS DASHBOARD

A strategic tool for hosts and investors to make data-driven decisions across Sydney, Paris, and Bangkok.

01 Project Introduction

This project provides a comprehensive overview of the Airbnb market across major global cities. The dashboard serves as a strategic tool for hosts and investors to analyze pricing optimization and customer behavior.

Pricing Optimization

Adjust rates based on geography & amenities.

Superhost Profiling

Identify key traits of top performers.

Technical Stack

  • MS Excel (ETL Stage 1)
  • Power BI (ETL & Modeling)
  • Power Query (M Language)
  • DAX Expressions

Data Analytics Workflow

Step 01

Data Acquisition (Folder Connection)

Instead of loading individual files, I used the Connect to Folder method. When new data is added, hitting "Refresh" updates all visuals automatically.

Automation Optimized
Step 02

Excel Pre-processing (ETL Stage 1)

Applied nested SUBSTITUTE, TRIM, and CLEAN to sanitize data. Standardized all pricing to USD using dynamic formulas.

Step 03

Advanced Transformation & Modeling

Used Unpivot in Power Query to normalize the Amenities column. Established a Star Schema for deep-dive analysis.
